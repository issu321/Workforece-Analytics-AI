### 🐧 Linux / macOS
```bash
bash install.sh
```

### 🪟 Windows
```cmd
install.bat
```

### 🐳 Docker
```bash
docker build -t workforce-analytics .
docker run -p 5000:5000 workforce-analytics
```

### ⚙️ Manual Installation
```bash
pip install -r requirements.txt
python app.py
```
