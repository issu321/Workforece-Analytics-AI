## 📦 Installation Guide

</div>

<br>

### 🐧 Linux / macOS
```bash
bash install.sh
```

### 🪟 Windows
```cmd
install.bat
```

### 🐳 Docker
```bash
docker build -t workforce-analytics .
docker run -p 5000:5000 workforce-analytics
```

### ⚙️ Manual Installation
```bash
pip install -r requirements.txt
python app.py
```

## 📁 Project Structure

</div>

<br>

```
Workforece-Analytics-AI/
├── 📄 app.py                 # Main Flask application
├── 📁 static/                # CSS, JS, images, uploads
│   ├── 📁 css/               # Stylesheets (Neon Horizon theme)
│   ├── 📁 js/                # JavaScript (animations, particles)
│   ├── 📁 uploads/           # User CSV uploads
│   └── 📁 reports/           # Generated reports
├── 📁 templates/             # HTML templates
│   ├── 🏠 index.html         # Landing page
│   ├── 📊 dashboard.html     # Analytics dashboard
│   ├── 🔮 predict.html       # Attrition prediction
│   ├── 📈 forecast.html      # Workforce forecasting
│   └── 🔐 login.html         # Authentication
├── 📁 models/                # Trained ML models
├── 📁 data/                  # Sample datasets
├── 📄 requirements.txt       # Python dependencies
├── 📄 install.sh             # Linux/macOS installer
├── 📄 install.bat            # Windows installer
├── 📄 Dockerfile             # Docker configuration
└── 📄 README.md              # This file
```
